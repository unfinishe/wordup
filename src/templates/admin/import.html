Import Instructions

Supported File Types:

  • JSON Files (.json): Single chapter exports from WordUp
  • ZIP Files (.zip): Full backups containing multiple JSON chapter files

What Gets Imported:

  • Chapter information (name, source/target languages)
  • Vocabulary cards (words, translations, example sentences)
  • Learning progress (box levels, review schedules)
  • Review history (past learning sessions and results)

Important Notes:

  • Chapters with identical names and language pairs will be skipped to prevent duplicates
  • All imported data will be added to your existing vocabulary
  • Review history will preserve original learning dates
  • Large imports may take a few moments to process
