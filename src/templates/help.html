Getting Started

1. Create a Chapter

Organize your vocabulary by creating chapters (e.g., "Basic German", "Travel Phrases"). Each chapter represents a topic or lesson.

2. Add Vocabulary Cards

Add word pairs with source word, translation, optional example sentences, and context hints. Use bulk import for faster entry.

3. Start Learning

Begin learning sessions to practice your vocabulary. WordUp uses spaced repetition to optimize your learning.

What are Context Hints?

Context hints are short descriptive notes that help you understand the relationship or nature of a word pair. They appear during learning sessions to provide additional context.

Common Context Types:
  • opposite - For antonyms (laut ‚Üí quiet)
  • synonym - For similar meanings (schnell ‚Üí fast)
  • formal - For formal language (Sie ‚Üí you)
  • informal - For casual language (Hey ‚Üí Hi)
  • plural - For plural forms (B√ºcher ‚Üí books)
  • verb form - For specific conjugations (bin ‚Üí am)

How to Use Context Hints

  1. Creating Cards: Add hints in the "Context/Hint" field when creating or editing cards
  2. Bulk Import: Include hints as the 4th column: word | translation | example | hint
  3. Learning: Hints appear automatically during practice sessions
  4. Visual Design: Hints are displayed with a lightbulb icon in a subtle blue style

  • Keep hints short and descriptive
  • Use consistent terminology across cards
  • Focus on relationships or word properties
  • Don't repeat information already in examples

The Leitner System

WordUp uses the scientifically proven Leitner System for spaced repetition learning.

How It Works:

  • Correct Answer: Card moves to the next box (longer interval)
  • Wrong Answer: Card returns to Box 1 (daily review)
  • Optimal Timing: Cards are reviewed just before you forget them
  • Efficient Learning: Spend more time on difficult words, less on easy ones
